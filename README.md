# hthpeer-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存放，不依赖于任何特定域名或后端服务。

## 项目简介

`hthpeer-journal` 是一个轻量级的页面归档仓库。你可以在此查看、分享或引用其中存放的 HTML 文档。仓库本身不承载任何动态内容，所有页面均为静态文件。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放主要 HTML 页面的文件夹
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── styles.css
│   ├── script.js
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有归档的 HTML 页面均放置于此目录下，按需命名。
- `assets/`：存放页面共用的样式、脚本及图片资源，便于复用与维护。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可单独通过浏览器打开。
- 页面内容不指向特定网站或服务，仅作为归档资料呈现。
- 如需引用某个页面，请使用该文件在仓库中的原始路径或 GitHub Pages 的对应地址。

## 维护说明

- 欢迎通过 Pull Request 提交新的归档页面，请确保页面内容符合主题且为静态 HTML。
- 提交前请确认页面内无外部依赖或已将所有依赖（如 CSS、JS）内联或放入 `assets/` 目录。
- 避免在页面中包含敏感信息或违反平台政策的内容。
- 本仓库不提供动态更新或通知机制，所有更新以提交记录为准。
